TV Video Streaming

TV App Design Remote Controlled

DETAILS

The objectives of this project was to design a TV video streaming application using brand guidelines from Frontify. The challange for this project was to design for a TV screen so I did a lot of research on different video streaming application and observed the typography, spacing, and layout of each in order to take what works with me into my design. Another challenge of this project was to create the design knowing that the user can only control it with the remote control. I was able to successfully do this by making sure it was easy to navigate from the navigation to the screen. One important factor that the client wanted was for the layout to be interesting and not boring. I believe I met that by changing up the video cards in each section. I was able to make it look aesthetically pleasing, functional, and interesting.
