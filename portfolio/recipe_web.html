Recipe Website

Desktop | iPad | Mobile

DETAILS

The objective of this project was to design a recipe website that could react responsively to mutiple screen sizes including desktop, ipad, and mobile phones. Keeping typography and type hierarchy in mind, I was able to effectively place type on all screens. I also chose shades of the colors green and pink to convey a natural healthy feel to the whole website. I also created interesting layouts and chose clear photographs to keep the user engaged. Overall, I believe this project was a success!
