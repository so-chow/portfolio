Smart Home Dashboard

Dashboard | iPad

DETAILS

The objective of this project was to create a beautifully functioning smart home dashboard that isn't too overwhelming for the user. The user should be able to see and change the inside temperature. The thermostat should have the ability to switch to heat and cold modes. The user should be able to see and control music and the different speakers in each room. There should be an ability to control the lights in each room and control the different security cameras throughout the house. The organization, color, and typography of the smart home was up to me. The most important element the client expressed was flexibility. For example, the flexibility to be able add as many light and speakers as you can so I made sure that flexibility was important in my design. I decided to go with a dark mode so as to not cause strain on the user's eye. I went with a color palette that is modern and cool. Purple and black are the main colors with a sprinkle of neon colors to escalate the look. I am really happy with my design and the mockup I took so long to make works really well. Solving for this was fun.
