Voting Website

Desktop | Mobile

DETAILS

The objective of this project was to design a website that caught the attention of young voters to encourage them to vote and inform them of all that they need to know to make it easy and simple to vote. I was tasked to design a website that could react responsively to mutiple screen sizes including desktop and mobile phones. Keeping typography and type hierarchy in mind, I was able to effectively place type on all screens. I chose shades of the colors purple and yellow in order to create a young hip vibe. I also created interesting layouts and eye catching illustrations to keep the user engaged. Overall, I believe this project was a success and this is one of my favorite websites I've designed.
